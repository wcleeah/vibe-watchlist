# Data

## Problem statement
We now have a bunch of events / data we can use, but no way to turn them into meaningful data.

## Details
- I want the following data, in three dimentions, average, total, per platform:
  - Token usage
  - Suggestion Acceptance rate
  - Video added
  - Video watched
  - Tag distribution
- And some particular data over a time range:
  - Hot hours for adding / watching videos
  - Tendency on watching videos of a tag
- All these data should be available in any time range (down to hours)
