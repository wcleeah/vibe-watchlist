# Extension

## Problem statement
The user has to go to the page to view / add video, not the ideal workflow. the ideal workflow should be having a chrome extension, that skip to the next video when done

## Details
- Develop a chrome extension, that make use of the current data structure
- It can:
  - Auto play: Detect when a video is done, navigate to the next url
  - Auto watched: When a video is done, mark it watched
  - Skip: allow user to skip to another video
  - Add video: allow user to add the current page as a to watch list, or paste a url to add
